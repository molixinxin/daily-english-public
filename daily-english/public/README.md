# Public Share Site

这个目录用于放“可公开分享”的英语学习网页。

## 你要分享给别人时

不能分享本地路径，例如 `E:\...` 或 `file:///...`，别人电脑访问不到。

你需要把 `daily-english/public/` 整个目录上传到任意静态托管服务，然后分享生成后的 `https://...` 网页地址。

## 最简单的托管方式

1. GitHub Pages
2. Netlify Drop
3. Cloudflare Pages

## 当前入口

- 首页：`index.html`
- 今天页面：`2026-06-24.html`

后续每天如果要公开分享，可以继续把当天页面复制到这个目录，并让 `index.html` 指向最新日期。
