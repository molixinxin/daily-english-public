# 项目级生成规则

## 英语学习材料

- 每日英语学习内容以 HTML 页面形式生成，并持续维护累计词汇 Excel。
- 所有中文字体统一使用 `Microsoft YaHei` / `微软雅黑`；HTML 和可视化材料中的中文说明、解释、练习、按钮、注释等都应显式使用该字体。
- 我通过邮件发送的中文内容也要使用 `Microsoft YaHei` / `微软雅黑`；如发送 HTML 邮件，应在正文或主要容器中设置 `font-family: "Microsoft YaHei", Arial, sans-serif;`。
- 英文标题、英文例句、诗歌或名言可根据版式使用 Georgia、Times New Roman、Aptos 等合适英文字体，但不得影响中文内容的微软雅黑显示。
- 累计词汇 Excel 表头固定为：日期、单词、音标、解释、例句。
- 每日 HTML 页面还应同步生成一个 Windows 快捷方式 `.lnk`，保存到 `daily-english/shortcuts/YYYY-MM-DD - Open in Chrome.lnk`，目标程序为 Google Chrome，参数为当天 HTML 文件路径；每日发送时优先提供这个 Chrome 快捷方式链接，再提供原始 HTML 链接。
- 每日 HTML 中每个新增单词本身都应支持点击发音；仅单词触发发音，音标和例句不需要发音。优先使用浏览器 Web Speech API，英文发音语言设置为 `en-US`，语速略慢于默认值，便于跟读。
- 每日摘录不需要与当天词汇主题一致；它的定位是高质量英文审美输入。优先选择文学、哲学、诗歌、散文、思想性名言中的高级表达，语言要有美感、密度和可反复品读的价值。避免过于基础、鸡汤化、口号化的句子。摘录应保持简短，优先使用公版文本或短句，并配中文赏析和 1 个可仿写的高级表达。
